Bienvenue !

Voici comment configurer l'application : 

  • Pour obtenir votre Client ID et le Client Secret Twitch, rendez-vous sur ce site : Twitch Developers.
  • Allez dans l'onglet "Applications" et enregistrez votre application.
  • Donnez-lui le nom que vous souhaitez. Entrez "http://localhost" dans le champ OAuth Redirect URLs, et sélectionnez la catégorie "Chat Bot", et le Client Type "Confidential".
  • Le Client ID s'affichera. Entrez-le dans le champ "Client ID Twitch" de l'application.
  • Cliquez ensuite sur "New Secret" pour générer un nouveau Client Secret. Attention, le client secret est une information sensible, ne le divulguez jamais.
  • Pour obtenir votre mot de passe OBS WebSocket : C'est celui qui est défini dans les paramètres OBS (plugin WebSocket). Attention, si votre version d'OBS Studio est inférieure à la 28, alors il faudra préinstaller le WebSocket.

Vous pouvez maintenant ajouter des sources de scène avec des commandes et les activer via le chat Twitch !
