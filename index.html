Bienvenue !

Voici comment configurer l'application : 

  • Pour obtenir votre Twitch Client ID et le OAuth Token, rendez-vous sur ce site : Twitch Token Generator.
  • Sélectionner les permissions dont votre bot aura besoin (a minima "chat:read" et "chat:edit"), puis validez.
  • La page devrait se recharger et afficher trois champs remplis : "Access Token", "Refresh Token" et "Client ID". Rentrez la clé générée pour "Access Token" dans "OAuth Twitch", et celle du "Client ID" dans "Client ID Twitch".
  • Pour obtenir votre mot de passe OBS WebSocket : C'est celui qui est défini dans les paramètres OBS (plugin WebSocket). Attention, si votre version d'OBS Studio est inférieure à la 28, alors il faudra préinstaller le WebSocket.

Vous pouvez maintenant ajouter des scènes avec des commandes et les déclencher via le chat Twitch !
